Movie API

This is a movie api that can be used to access a list of movie detils including: Movie Title, Genre, Release Year, Actors and description.

Request URL Response
List all movies to user /movies List of all available movies with image and title
Return available data about specific movie /movies/[movie title] Results in movie title, cover art, description, genre, featured film, director, actors
Return a list of movies by genre /movies/genres/[genre] Returns and a list of movies that match the genre
Return Directors Bio /directors/[directors name] Gives information about specific director (bio, birth date, death year, featured work)
Sign up a new user /sign-up Allows a user to register a new account
User login /login User can login to their account
View users info /users/[username] User can view their account details
Delete account /users/[username]/delete Allows a user unregister and delete their account
Update a users info /users/[user-name]/edit Update a users inforamtion
Add to favorites users/[username]/favorites/add-favorites Let's user add a movie to their favorites list
Remove from Favorites /users/[username]/favorites/remove-favorites User can remove item from their favorites list
